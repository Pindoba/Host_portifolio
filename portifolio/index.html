Como hobby já hospedei alguns sites em servidores gratuitos, para adquirir mais conhecimentos em configurar banco de dados SQL, conexão cliente /servidor, e afins.

Comanda pré-paga

Projeto pensado para casas noturnas e bares, onde o cliente recebe um cartão com o codigo único podendo adicionar valor para o consumo no resínduo, e ao final da noite recupar o valor restante. Podendo tambem ser ultilizado como uma "carteirinha fidelidade". Tudo fica registrado em uma tabela sql, inclusive todas as movimentações de entrada e saída.

Hospedagem

Um simples projeto de hospedagem de uma aplicação de conversação (salas de bate-papo). Lembrando que nesse caso eu não desenvolvi a aplicação, eu hospedei e configurei em um servidor publico. Fazendo sua instalação, configuração do banco de dados, domínio, instalação de certificado SSL, etc.

Fila Karaokê

Uma aplicação desktop simpres construida em python com interface utilizando a biblioteca pyqt5 e banco de dados sqlite. Onde sua função é gerenciar uma fila de pessoas a cantar no karaoke, incluindo o nome na fila, editando e excluindo. Salvando todo o histórico de quem ja cantou.
